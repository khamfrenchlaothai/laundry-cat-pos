## üÜò Troubleshooting

### "Your site is not published yet"
- Wait 2-3 minutes and refresh
- Check that you selected **main** branch
- Ensure repository is **Public**

### "404 Not Found"
- Make sure `index.html` is in the root folder
- Check the URL is correct
- Wait a few more minutes

### App won't install on tablet
- Use Safari (iOS) or Chrome (Android)
- Make sure you're on the GitHub Pages URL (not github.com)
- Try clearing browser cache

### Changes not showing
- Wait 2-3 minutes after uploading
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Clear browser cache
