## 1Ô∏è‚É£ Making a Sale (POS)

### Step-by-Step

1. **Tap "POS"** in the left sidebar
2. **Search or browse** for items
3. **Tap an item** to add it to the cart
4. **Adjust quantity** using + and - buttons
5. **(Optional) Add customer**:
   - Tap "Add Customer to Sale"
   - Search by name or phone
   - Select customer
6. **Tap "Charge"** button
7. **Select payment method**: Cash, One Pay, or QR
8. **Choose to print ticket** (checked by default)
9. **Tap "Confirm"**
10. **Ticket opens** in new window - tap "Print" if needed

‚úÖ Sale complete!

## 2Ô∏è‚É£ Managing Items

### View All Items

1. Tap **"Items"** in sidebar
2. See all services and products
3. Use search to find specific items

### Add New Item

1. Tap **"+ Add Item"** button
2. Fill in details:
   - **Name**: e.g., "Wash & Fold"
   - **Price**: in LAK (e.g., 15000)
   - **Category**: Laundry, Dry Cleaning, Bedding, or Retail
   - **Image URL** (optional): Link to image
3. Tap **"Save"**

### Edit Item

1. Find the item in the list
2. Tap **"Edit"** button
3. Change details
4. Tap **"Save"**

### Delete Item

1. Find the item
2. Tap **"Delete"** button
3. Confirm deletion

‚ö†Ô∏è **Warning**: Deleted items cannot be recovered!

## ‚ö†Ô∏è Common Issues

### Cart is stuck
- Tap "Clear" button

### Item won't add to cart
- Check if item still exists in Items section
- Try refreshing the page

- Export and save your data
- Delete old sales from dashboard
- Clear browser data and re-import
